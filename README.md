## Helpful Commands

Build:

- `sudo docker build -t cute-puppers:latest .`

Run:

- `sudo docker run -p 5000:5000 cute-puppers`

Save conda env (don't forget to remove the prefix):

- `conda env export > bin/environment.yml`
